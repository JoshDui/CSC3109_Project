# Reports

Use this folder for report materials and generated figures.

Suggested structure:

```text
reports/
  figures/
  tables/
  drafts/
```

Keep final report evidence here:

- Dataset summary tables.
- EDA figures.
- Training curves.
- Confusion matrices.
- Model comparison tables.
- Streamlit and Docker screenshots.

The canonical EDA notebook (`notebooks/01_dataset_eda.ipynb`) writes the main
EDA artifacts to this folder.
